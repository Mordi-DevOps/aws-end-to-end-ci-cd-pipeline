# aws-end-to-end-ci-cd-pipeline
End-to-End AWS CI/CD Pipeline Portfolio Project, using Jenkins, Docker, and Terraform
---------------------------------------------------------------------------------------
This repository contains a complete DevOps portfolio project demonstrating an end-to-end CI/CD pipeline on AWS.
It includes:
- Infrastructure as Code (Terraform) for AWS resources
- Jenkins CI/CD pipeline setup
- Docker containerization for application and services
- Architecture diagrams and screenshots
